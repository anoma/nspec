# LocalComputeSearchV1

## Purpose

<!-- ANCHOR: purpose -->
Perform a search operation based on a given predicate within a specified time limit (timeout).
<!-- ANCHOR_END: purpose -->

## Type

<!-- ANCHOR: type -->
<div class="type">

*Records* with fields:

  *Predicate describing the search operation*

- `Timeout`: [[TimeoutV1#timeoutv1]]
