# SuccessOperationV1

## Purpose
<!-- --8<-- [start:purpose] -->
Indicates whether an operation performed by [[Local Key-Value Storage Engine#local-key-value-storage-engine]]
was executed successfully or not. 

<!-- --8<-- [end:purpose] -->

## Type

<!-- --8<-- [start:type] -->
<div class="type">


</div>
<!-- --8<-- [end:type] -->

## Values

Boolean value where:
- `1`: Operation was successful. 
- `0`: Operation was unsuccessful.
