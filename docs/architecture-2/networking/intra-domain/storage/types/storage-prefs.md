# StoragePrefs

## Purpose

<!-- ANCHOR: purpose -->
Storage preferences.
<!-- ANCHOR_END: purpose -->

## Type

*Struct* with the following fields.

| Field       | Type        | Description                                                           |
|-------------|-------------|-----------------------------------------------------------------------|
| `expiry`    | *Timestamp* | Expiry time after which the block is deleted.                         |
| `min_trust` | *u8*        | Minimum trust value required when serving the block to third parties. |
