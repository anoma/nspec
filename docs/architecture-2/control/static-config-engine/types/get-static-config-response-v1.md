# GetStaticConfigResponseV1

## Purpose

<!-- ANCHOR: purpose -->
Return the found value corresponding to key in  [[GetStaticConfigRequestV1#getstaticconfigrequestv1]]. 
<!-- ANCHOR_END: purpose -->

<!-- ANCHOR: type -->
<div class="type">

  *The requested static configuration value.*
