# CommitRequest

## Purpose

<!-- ANCHOR: purpose -->
A `CommitRequest` instructs a commitment engine instance to produce a commitment by the internal identity corresponding to that engine instance.
<!-- ANCHOR_END: purpose -->

## Type

<!-- ANCHOR: type -->
<div class="type">
*Record* with fields:

- `data`: *[[Signable]]*

  *Data over which to commit*
</div>
<!-- ANCHOR_END: type -->
