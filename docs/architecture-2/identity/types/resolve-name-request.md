# ResolveNameRequest

## Purpose

<!-- --8<-- [start:purpose] -->
A `ResolveNameRequest` asks the name engine which [[ExternalIdentity]]s are associated with a given [[IdentityName]].
<!-- --8<-- [end:purpose] -->

## Type

<!-- --8<-- [start:type] -->
<div class="type" markdown>

*Record* with fields:

- `name`: [[IdentityName]]

  *Name to query*
</div>
