# DecryptRequest

## Purpose

<!-- ANCHOR: purpose -->
A `DecryptRequest` instructs a decryption engine instance to decrypt data as the internal identity corresponding to that engine instance.
<!-- ANCHOR_END: purpose -->

## Type

<!-- ANCHOR: type -->
<div class="type">
*Record* with fields:

- `data`: `[]byte`

  *Encrypted ciphertext to decrypt**
</div>
<!-- ANCHOR_END: type -->
