# VerifyResponse

## Purpose

<!-- ANCHOR: purpose -->
A `VerifyResponse` contains the result of verifying a commitment in response to a [[VerifyRequest]].
<!-- ANCHOR_END: purpose -->

## Type

<!-- ANCHOR: type -->
<div class="type">
*Record* with fields:

- `result`: boolean

  *Whether or not verification succeeded*
</div>
<!-- ANCHOR_END: type -->
