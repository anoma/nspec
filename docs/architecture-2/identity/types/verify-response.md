# VerifyResponse


## Purpose


<!-- --8<-- [start:purpose] -->
A `VerifyResponse` contains the result of verifying a commitment in response to a [[VerifyRequest]].
<!-- --8<-- [end:purpose] -->

## Type


<!-- --8<-- [start:type] -->
<div class="type" markdown>

*Record* with fields:

- `result`: boolean

  *Whether or not verification succeeded*
</div>
<!-- --8<-- [end:type] -->
