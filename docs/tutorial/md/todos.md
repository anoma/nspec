## Todos

Incorporate todos with the following syntax:


```text
!!! todo

    Content of the todo
```

The above renders as:

!!! todo

    Content of the todo

!!! info

    Be aware that todos are automatically removed from the online version. If you want to keep them, set `todos: True` in the front matter.
