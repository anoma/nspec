# Map

Map is a structure that contains pairs (key: value), where key is of type `K` and value is of type `V`.

## Interface

1. `New() -> Map`
2. `Add(Map, K, V) -> Map`
3. `Size(Map) -> Nat`
4. `Get(Map, K) -> V`
5. `Keys(Map) -> List K`
